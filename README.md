# Silver Fox

Silver Fox is a customizable Mac-focused control center designed for seniors with low tech comfort.

## Current Prototype
- Elder View with bottom-anchored circular action dock
- Caregiver lock/unlock and button customization
- Instant-open actions (web/call/message/app/automation)
- Center time/date display
- High-contrast and reduced-motion options

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
